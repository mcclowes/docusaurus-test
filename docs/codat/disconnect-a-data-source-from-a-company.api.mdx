## Disconnect a data source from a company



This revokes a company’s access to a data source, but the data connection is still listed as a part of a company&#39;s
data connection(s). The status value in the request body should be &quot;Unlinked&quot; (case sensitive). Data connections
can only be unlinked if in the Linked or Deauthorized state.
